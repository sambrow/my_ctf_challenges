You should have gotten these two JSON outputs from decoding the first two regions of the JWT: 
    {"alg":"HS256","typ":"JWT"}
    
    {"username":"sam","isAdmin":false,"iat":1738887694,"exp":1738889494}

The first JSON conveys which cryptographic signing algorithm was used to create the token.

The second JSON contains the "claims". It says the username is sam and that they are not an admin. iat stands for "issued at" and exp stands for "expiration". These are date/time stamps but we won't worry about these for this challenge.
