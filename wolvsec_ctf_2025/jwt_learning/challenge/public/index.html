Welcome to the JWT Learning challenge. Over the course of the challenge, we'll try to teach you a little about this topic.

JWT stands for JSON Web Token.

This is a digitally signed token that carries information called "claims" about the person who just logged into the system. It is just a bunch of text in a special format.

Here is what one looks like: 
        eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VybmFtZSI6InNhbSIsImlzQWRtaW4iOmZhbHNlLCJpYXQiOjE3Mzg4ODc2OTQsImV4cCI6MTczODg4OTQ5NH0.KrRwMea_fXtfp-IbvFCn3Q-HiNVMitips0SVxODfGJc

If you look carefully, you'll see this has three regions separated by two periods. 
  • eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9
  • .
  • eyJ1c2VybmFtZSI6InNhbSIsImlzQWRtaW4iOmZhbHNlLCJpYXQiOjE3Mzg4ODc2OTQsImV4cCI6MTczODg4OTQ5NH0
  • .
  • KrRwMea_fXtfp-IbvFCn3Q-HiNVMitips0SVxODfGJc

The first two regions are base64 encoding of JSON. JSON is a way Javascript represents data. Here is an example simple JSON object: 
        {"color": "blue"}

To better understand this, let's practice doing this encoding ourself. Copy the above JSON to the clipboard and then go to this page.

Paste the JSON into the Text* window and then click the Encode to Base64 button. You should see this appear in the Base64 output window: 
eyJjb2xvciI6ICJibHVlIn0=

To understand that this is an encoding and not an encryption, let's decode it. Copy the base64 string to the clipboard and then go to this page.

As before, paste it in and click the button. You should see the original JSON come back. That proves that this is just a different way of representing the original characters and that it is completely reversible without needing any kind of password.

Of interest is the = symbol at the end. This is something called "padding" but is not strictly needed by most modern base64 decoders. In fact, you can remove the = from your example and click the button and you'll get the exact same JSON string back.

Now, use what you've learned to copy each of the first two JWT regions into the base64 decoder page and see what comes out.
